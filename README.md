# ResuFlow: AI-Driven Portfolio Pipeline

**ResuFlow** is an automated pipeline designed to transform static CVs into hosted, high-conversion professional portfolios. It bridges the gap for non-technical professionals by using AI to extract data from a resume and deploy it as a personalized, web-accessible portfolio.

---

## 🚀 Features

* **AI Extraction Engine**: Automatically parses PDF or Docx files to map unstructured text into structured JSON profiles.

* **Zero-Code Personalization**: Users can select "Vibes," color palettes, and layout archetypes without any technical skills.

* **Automated Cloud Hosting**: Programmatic deployment to secure URLs via Vercel or Netlify APIs.

* **QR Code Integration**: Generates high-resolution (Level H) QR codes to link physical CVs to digital portfolios.

* **Schema Validation**: Ensures all extracted data conforms to system standards for consistency and accuracy.

---

## 🛠️ Tech Stack

### Frontend

* **React**: Used for building interactive dashboards and supporting server-side rendering for responsive portfolio templates.

### Backend

* **Node.js & Express**: Provides a fast, event-driven environment for handling file uploads, API requests, and data transformation.

### Database

* **MongoDB**: Stores user profiles and configuration choices in a flexible, schema-less JSON format.

### AI & Infrastructure

* **Gemini API**: Powers the intelligence layer to identify skills, work history, and achievements from raw text.


* **Vercel / Netlify**: Handles automated hosting and continuous integration for the generated portfolios.
